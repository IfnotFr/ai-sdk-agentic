# AI SDK Agentic - Documentation Landing Page

This is a temporary, high-tech landing page for the AI SDK Agentic framework. It is built with Vite and vanilla CSS.

## Getting Started

1. Go to the root of the monorepo:
   ```bash
   npm install
   ```

2. Run the development server for the docs:
   ```bash
   npm run dev -w ai-sdk-agentic-docs
   ```

## Deployment

This app is ready to be deployed on Vercel or any other static hosting provider.
The `vercel.json` ensures that it behaves well as a Single Page Application if needed.
